研究

研究方向

零界演化（SingularityX Evolution）致力于构建以 AI 自进化为核心的智能决策系统，并以金融市场这一高度非平稳、强对抗环境作为真实实验场。研究重点并不止于“更好的预测模型”，而是指向更本质的问题：智能体如何在动态环境中形成稳定决策能力，多智能体博弈中策略如何产生、竞争与进化，以及如何由“人类设计策略”进一步迈向“系统自主演化策略”。

推动决策系统由人工设计范式向自主进化范式演进

系统框架

世界模型 + 多 Agent 博弈 + 强化学习 = 可自进化的智能决策系统

演化闭环

真实市场 + 模拟世界 + 自博弈筛选 = 策略的持续生成、竞争与进化

金融市场并非终点，而是验证复杂智能决策系统的重要实验场。

核心研究问题

  • 如何使智能体在非平稳环境中形成稳定学习机制
  • 如何构建具备可收敛性的策略演化系统
  • 如何将博弈机制、强化学习与真实市场反馈有效连接

岗位特征

  • 工作对象并非标准化需求，而是尚未定型的核心问题
  • 需要在研究、工程实现与实验验证之间保持高频切换
  • 岗位定位并非流程执行，而是面向能够推动系统持续迭代的研究与工程人才

招募

开放岗位

欢迎 ACM / 算法 / 强化学习 / 多 Agent / 世界模型方向的工程师与研究员加入。

ACM / 算法工程师
Search / Optimization / Systems

职责：设计高性能搜索、调度、推演与优化模块，支撑训练、仿真与策略生成。

要求：ACM / ICPC、OI 或强算法背景优先；数据结构、图算法、优化与系统性能扎实。

多 Agent 强化学习研究员
Multi-Agent RL Researcher

职责：研究非平稳环境下的策略学习、协同/竞争机制与自博弈训练方法。

要求：熟悉 RL / MARL / Self-Play，有论文、竞赛或项目落地经验者优先。

世界模型研究员（金融环境）
World Model Researcher

职责：构建市场环境抽象、状态表示、模拟器与可泛化的推演框架。

要求：熟悉时序建模、生成式模型、世界模型或仿真系统，对复杂环境建模有深入兴趣。

策略演化系统研究员
Evolutionary Strategy Systems Researcher

职责：搭建可收敛的策略演化系统，打通研究实验、回测、仿真与真实市场。

要求：能够打通理论、代码实现与实验闭环，重视严谨性、可解释性与执行力。

岗位对应的是面向真实环境、真实反馈与真实责任的研究与工程场景，以及参与定义下一代智能决策系统的机会。
